# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Стек технологий

- **Python 3** — основной язык разработки
- **Jinja2** — шаблонизатор для генерации HTML
- **Pandas** — обработка данных из Excel
- **openpyxl** — движок для работы с `.xlsx` файлами
- **HTTP Server** — встроенный модуль Python для запуска локального сервера

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <repo_url>
   cd wine
   ```

2. Создайте виртуальное окружение и активируйте его:
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # для Linux/Mac
   # venv\Scripts\activate   # для Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirepment.txt
   ```

4. Запустите скрипт генерации сайта и сервер:
   ```bash
   # Запуск с файлом по умолчанию (wine.xlsx)
   python3 main.py

   # Запуск с указанием пользовательского файла данных
   python3 main.py --excel path/to/your_data.xlsx
   ```
   По умолчанию используется файл `wine.xlsx` из текущей директории.

5. Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).

## Возможности

- Каталог вин, разделенный по категориям (Белые вина, Красные вина, Напитки)
- Детальная информация о каждом продукте: название, сорт винограда, цена, изображение
- **Акции:** Продукты, помеченные в файле данных (по умолчанию `wine.xlsx`, настраивается через `--excel`) как "Выгодное предложение", отображаются со специальным значком
- Автоматический расчет возраста компании (с 1920 года)

## Структура данных

Проект использует Excel-файл (по умолчанию `wine.xlsx`, можно изменить через аргумент `--excel`) со следующими колонками:
- **Категория** — категория вина
- **Название** — название продукта
- **Сорт** — сорт винограда
- **Цена** — стоимость продукта
- **Картинка** — имя файла изображения
- **Акция** — пометка для отображения значка "Выгодное предложение"
